# Superstore Sales Data Visualization

## Project Overview

This project transforms raw sales data into meaningful visualizations to identify trends, patterns, and business insights. Various charts and graphs are used to support decision-making and improve understanding of sales performance.

## Objectives

- Transform raw data into visual formats.
- Analyze sales and profit trends.
- Identify high-performing regions and categories.
- Create data-driven business insights.
- Build an impactful visualization portfolio project.

## Tools & Technologies

- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Dataset Information

- Rows: 2000
- Columns: 12

## Visualizations

- Sales by Category
- Profit by Category
- Sales by Region
- Customer Segment Analysis
- Discount vs Profit
- Sales Trend Analysis
- Shipping Mode Analysis
- Correlation Heatmap

## Key Insights

- Technology products generated strong sales.
- Higher discounts reduced profitability.
- Some regions performed better than others.
- Customer segments contributed differently to sales.

## Conclusion

The visualizations successfully transformed raw sales data into meaningful business insights and supported better decision-making.
